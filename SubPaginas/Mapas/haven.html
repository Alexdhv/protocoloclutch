Estrategias Básicas

Ataque al Sitio A

Ruta Principal: Ruta Principal: A Long → A Site

Alternativa: A Short (por Garage) → A Site

Consejo: Controlar Garage y A Lobby es clave para evitar flanqueos.

Ataque al Sitio B

Ruta Principal: Mid → B Site (usando ventana o puertas)

Alternativa: A Short → B Link → B Site

Consejo: Usar smokes en Mid para evitar que los defensores te vean desde C o A.

Ataque al Sitio B

Ruta Principal: Mid → B Site (usando ventana o puertas)

Alternativa: A Short → B Link → B Site

Consejo:Usar smokes en Mid para evitar que los defensores te vean desde C o A.

Ataque al Sitio C

Ruta Principal:C Long → C Site

Alternativa: C Short (por Spawn defensivo) → C Site

Consejo: Vigilar C Long desde posiciones elevadas (como el cubo en C Site).

Defensa General

Puntos clave: Controlar Mid y Garage para evitar rotaciones rápidas.

Rotaciones: Usar B Link y A Short para moverse entre sitios.

Consejo: Un jugador debe vigilar Mid para detectar pushes hacia B.

Consejos

  • Mid controla el mapa. Si el equipo atacante domina Mid, puede presionar A, B o C fácilmente.
  • Jett y Omen son fuertes en este mapa por sus habilidades para tomar altura s
  • El sitio B es el más expuesto - Usa smokes o flashes para plantar allí.
  • Las rotaciones defensivas son más largas - Los atacantes pueden explotar esto con fakes.
