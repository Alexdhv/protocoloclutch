Estrategias Básicas

Ataque al Sitio A

Ruta Principal: A Short → Teletransportador → A Site

Alternativa: A Long → A Lobby → A Site

Consejo: Controlar Showers es clave para dividir la defensa enemiga

Ataque al Sitio B

Ruta Principal: B Long → B Site

Alternativa: Hookah → B Site

Consejo: Usar smokes en B Long es esencial para el avance

Defensa General

Puntos clave: Controlar Hookah y Showers

Rotaciones: Usar teletransportadores para movimientos rápidos

Consejo: Colocar trampas o alarms en los TP para detectar enemigos

Consejos

  • Los teletransportadores hacen ruido - úsalos con cuidado
  • Jett y Omen pueden usar sus habilidades para acceder a posiciones elevadas
  • El sitio B tiene menos cubierta que el A - considera esto al plantar
  • Controlar U-Hall es crucial para defender el sitio A
  • Las habilidades de humo son especialmente valiosas en este mapa
